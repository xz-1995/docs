
# markdown语法

## 标题

```md
# 这是一级标题
## 这是二级标题，二级标题底下有横线
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

## 字体

```md
加粗: **这里是加粗字体**
斜体: *斜体*
删除: ~~删除的字体~~
```

## 引用

```md
>这是1级引用的内容

>>这是2级引用的内容

>>>这是3级引用的内容
```

## 分割线

```md
***
---
```

## 图片

```md
格式: ![图片alt] (图片地址 “图片title”)
例如: ![风景](/public/images/888.jpeg "风景")
```
::: tip
public 文件夹在项目根目录
:::

## 代码分组


\::: code-group  
\```js [1.js]  
const a = 1  
\```  
\```js [2.js]  
const a = 2  
\```  
\:::
